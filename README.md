# Sarmad Global Website

Static website for Sarmad Global, a UK accountancy and practice modernisation firm.

## Structure

- `index.html` - Home page
- `about.html`, `pricing.html`, `resources.html`, `contact.html` - Core pages
- `businesses/` - Business and individual service pages
- `practices/` - Accounting practice service pages
- `legal/` - Legal information pages
- `css/style.css` - Site styles
- `js/app.js` - FAQ, calculator, and upload interactions

## Deployment

This site is static and can be hosted on GitHub Pages, Hostinger static hosting, or any standard web server by uploading the repository contents to the web root.
